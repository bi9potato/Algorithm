# Code

```java

class Solution {
    public List<Integer> findDuplicates(int[] nums) {

        for (int i = 0; i < nums.length; i++) {
            while (nums[i] != nums[nums[i]-1]) {
                swap(nums, i, nums[i]-1);
            }
        }

        List<Integer> res = new LinkedList<>();
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != i+1) {
                res.add(nums[i]);
            }
        }

        return res;
        
    }

    public void swap(int[] nums, int i, int j) {
        int tmp = nums[i];
        nums[i] = nums[j];
        nums[j] = tmp;
    }
}

```

# Complexity Analysis
- Time $\mathcal{O}(N)$
- Space $\mathcal{O}(1)$
